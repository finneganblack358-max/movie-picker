# Movie Picker

## Installation
1. Clone the repo: `git clone https://github.com/finneganblack358-max/movie-picker`
2. Install requirements.txt: Windows: `pip install -r requirements.txt` Mac/Linux: `pip3 install r requirements.txt`

## How to run
1. In repo directory input: Windows: `python main.py` Mac: `python3 main.py`
2. Click on link
